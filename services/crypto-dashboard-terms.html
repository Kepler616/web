Crypto Real-Time Admin Dashboard demo terms of use

Términos de uso del demo Crypto Real-Time Admin Dashboard

This page describes the scope and limitations of the crypto dashboard demo available on this site.

Esta página describe el alcance y las limitaciones del demo de dashboard cripto disponible en este sitio.

Demo only

Solo demostración

The Crypto Real-Time Admin Dashboard is a personal engineering demo built to showcase backend and frontend integration, real time style polling, data normalization and UI design. It is not a production application.

El Crypto Real-Time Admin Dashboard es un demo personal de ingeniería diseñado para mostrar integración entre backend y frontend, un estilo de actualización en tiempo casi real, normalización de datos y diseño de interfaz. No es una aplicación de producción.

No financial advice

Sin asesoría financiera

The information displayed in the dashboard does not constitute investment, trading or financial advice. No decisions should be made based on the data shown in this demo.

La información mostrada en el dashboard no constituye asesoría de inversión, trading ni financiera. No se deben tomar decisiones basadas en los datos presentados en este demo.

Data source and limitations

Fuente de datos y limitaciones

Market data is obtained from a third party public API through a Cloudflare Worker backend. Values may be delayed, incomplete or unavailable. The demo is intentionally limited to a small curated list of assets and a refresh interval of 10 seconds.

Los datos de mercado se obtienen de una API pública de terceros a través de un backend en Cloudflare Workers. Los valores pueden estar retrasados, ser incompletos o no estar disponibles. El demo está limitado de forma intencional a una lista pequeña de activos seleccionados y a un intervalo de actualización de 10 segundos.

Reliability and availability

Confiabilidad y disponibilidad

Because this is a personal demo, uptime, error handling and resilience are designed to illustrate engineering approaches rather than guarantee continuous service. The dashboard may be modified, paused or removed at any time.

Debido a que se trata de un demo personal, la disponibilidad, el manejo de errores y la resiliencia están diseñados para ilustrar enfoques de ingeniería y no para garantizar un servicio continuo. El dashboard puede modificarse, pausarse o eliminarse en cualquier momento.

Any data accessed through this demo must be treated as illustrative. By using the dashboard you agree that the author is not responsible for any outcome related to the use, interpretation or availability of the information displayed.

Cualquier dato accedido a través de este demo debe tratarse como ilustrativo. Al usar el dashboard aceptas que el autor no es responsable por ningún resultado relacionado con el uso, la interpretación o la disponibilidad de la información mostrada.
